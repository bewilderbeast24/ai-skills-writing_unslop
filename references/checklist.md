# Checklist: unslop-ai Execution

- [ ] Read input text.
- [ ] Identify AI slop patterns using [slop-checklist.md](slop-checklist.md).
- [ ] Create roadmap for rewrite.

## Stage 2: Unsloping Rewrite

## Stage 3: Final Polish
- [ ] Verify against [final-verification.md](final-verification.md).
- [ ] Remove remaining em-dash addiction or bold-first bullets.
- [ ] Confirm natural flow and varied tone.
